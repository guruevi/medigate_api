# **get_devices**
> GetDevicesResponse get_devices(get_devices_parameters)

Get details of devices

Get details of devices from the inventory. The data returned by this endpoint corresponds to the Devices table in the dashboard.  Note that this endpoint will return all devices, including retired devices.
